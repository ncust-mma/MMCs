## Installation

```bash
$ git clone https://github.com/Zeroto521/MMCs.git
$ cd MMCs
$ pipenv install --dev
$ pipenv shell
$ flask forge
$ flask translate compile
$ flask run
* Running on http://127.0.0.1:5000/
```
